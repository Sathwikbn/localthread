# Sketch 001: Shop Showcase & Origin Mapping

## Design Question
How should we present independent local boutiques, craft ateliers, and neighborhood design studios on the platform, and map product origin clearly?

## How to View
Open: [index.html](file:///c:/Users/sathw/Desktop/lt/localthread/localthread/.planning/sketches/001-shop-showcase/index.html)

## Variants
- **A: Bento Discovery + Floating Badge** — An asymmetrical, bento grid layout to showcase store highlights dynamically. Individual items in the shopping grid display a small, elegant floating origin badge.
- **B: Cinematic Carousel + Hover Overlay** — A horizontal card carousel (Apple-style) that allows collectors to browse store stories like a sliding catalog. Hovering over a product reveals a detailed card containing store coordinates and story descriptions.
- **C: Split-Screen Explorer + Side list** — A multi-pane viewport containing a searchable boutique listing sidebar on the left and full collection spotlights on the right. Perfect for deep catalog discovery on desktop screens.

## What to Look For
*   **Discovery Feeling**: Compare whether the Bento grid, the carousel, or the split-screen feels easiest to browse.
*   **Product Mapping**: Compare whether the visible floating badge (Variant A) or the interactive hover detail card (Variant B) is better for communicating which shop sells the product.
*   **Interactivity**: Hover over elements, toggle follow buttons, open details drawers, use the filter inputs, and use the floating bottom-right sketch tool panel to test mobile/tablet viewport responsiveness.
